# RCS1199: Unncessary null check

| Property | Value      |
| -------- | ---------- |
| Id       | RCS1199    |
| Category | Redundancy |
| Severity | Info       |

## Examples

### Code with Diagnostic

```csharp
bool? x = null;

// ...

if (x.HasValue && x.Value) // RCS1199
{
}
```

### Code with Fix

```csharp
if (x == true)
{
}
```

- - -

### Code with Diagnostic

```csharp
bool? x = null;
bool y = false;

// ...

if (x != null && x.Value == y) // RCS1199
{
}
```

### Code with Fix

```csharp
if (x == y)
{
}
```
